# For Clock Hours
```
12 hours = 360deg
1 hour = 360/12 = 30deg
h hour = 30h + m/2
```

# For Clock Hours Only

```
60 min = 30deg
1 min = 30/60 = 1/2
m min = m/2
```

# For Clock Minutes

```
60 min = 360deg
1 min = 360/60 = 6deg
m minute = 6m
```

# For Clock Seconds

```
60 sec = 360deg
1 sec = 360/60 = 6deg
s second = 6s
```
